## SNHU - IT-511 Object-Oriented App Development Final Project
## Java Virtual World Appliction

This application gathers information about the user and repeats it back for confirmation. It then asks if the user has a dog and, if affirmative, gathers information about the dog and repeats it back for confirmation.

Finally, there is a shout box with canned phrases that can be picked or chosen at random. The application also allows the user to enter their own text for teh shoutbox.
